##### Ignite feed was made to improve many abilities necessary to an front-end software developer, such as :

1. Follow a design made in figma
2. Typescript in ReactJs
3. ReactJs concepts such as
   - Hooks
   - Dealing with state
   - Props
   - Interacting in JSX
   - Key in React
   - Form Validation
   - React Closures

> Funcionalities implemented

- Possible to add new comments
- Like the comment
- Possible to delete an comment
